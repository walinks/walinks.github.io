Technical Reference

WhatsApp Link Formats Explained

Understand the different wa.me URL formats and when to use each one for your click-to-chat links.

Basic Click-to-Chat Link

https://wa.me/[phone]

The simplest WA.me link format. Opens WhatsApp with the specified number pre-loaded. No message is included. Best for business cards, email signatures, and simple contact buttons.

Link with Pre-filled Message

https://wa.me/[phone]?text=Hello%20there!

Adds a URL-encoded pre-filled message. When the user clicks, the message is already typed in the chat box — they just press send. Ideal for marketing, lead gen, and sales funnels.

WhatsApp API Format (wa.me)

https://api.whatsapp.com/send?phone=[phone]&text=Hi

The older API format still widely supported. Works identically to wa.me links. Both formats open WhatsApp on mobile apps and WhatsApp Web for desktop users.

Phone Number Format Rules

Country Code + Number (no + sign, no spaces)

Phone numbers must include the international country code. Remove all spaces, dashes, and the leading "0". Example: UK +44 7911 123456 becomes 447911123456 in the link.

Message URL Encoding

Spaces = %20, & = %26, ? = %3F

Messages in WhatsApp links must be URL-encoded. Our generator handles this automatically — just type your message in plain English and we encode it for you correctly every time.

WhatsApp QR Code Format

QR encodes the full wa.me URL

WhatsApp QR codes simply encode your wa.me link as a machine-readable image. When scanned with any camera app, it opens WhatsApp directly — perfect for print and physical media.

WhatsApp Links: Complete Guide

Learn how WhatsApp click-to-chat links work, the wa.me format, and best practices for using WA links in your business.

What Is a WhatsApp Link (WA.me)?

A WhatsApp link, also known as a click-to-chat link or WA.me link, is a URL that opens a WhatsApp chat window directly when clicked. The link format is https://wa.me/PHONENUMBER where the phone number includes the country code but no + sign.

When a user clicks a WhatsApp link on a mobile device, it opens the WhatsApp app. On desktop, it opens WhatsApp Web. The recipient doesn't need to save the number first — it jumps straight to the chat screen.

How to Create a WhatsApp Link for Free

Creating a WhatsApp link is simple with WALink Generator. Enter the phone number with country code, optionally write a pre-filled message, and click Generate. The tool instantly produces a valid wa.me URL you can copy, share, or embed anywhere.

For manual creation, use: https://wa.me/[country_code][phone]?text=[encoded_message]. Replace spaces in the message with %20 and special characters with their URL-encoded equivalents.

WhatsApp Link for Business & WhatsApp Business API

WhatsApp Business links work exactly the same as personal links — the wa.me URL format is identical. The difference is the type of account the number belongs to. Adding a WhatsApp link to your Google Business listing, website, and social media can significantly increase customer contact rates.

Businesses using the WhatsApp Business API can use the same wa.me link format. Pre-filled messages are especially powerful for businesses — they help qualify leads before the conversation even starts.

WhatsApp QR Code: What It Is & How to Use It

A WhatsApp QR code is simply a wa.me link encoded as a visual QR image. When scanned with a smartphone camera, it opens WhatsApp exactly like clicking the link. QR codes are perfect for print materials — business cards, flyers, restaurant menus, event banners, and product packaging.

WALink Generator creates high-resolution downloadable QR codes for any WhatsApp link. Just generate your link and click the QR Code button. Download the PNG and use it in any design application or print it directly.

Best Practices for WhatsApp Click-to-Chat Links

Always include a pre-filled message that sets context — it increases the chance the recipient knows what the conversation is about. For marketing, use specific, campaign-relevant messages like "Hi, I saw your Facebook ad about [product]."

Place WhatsApp links prominently — above the fold on websites, in email footers, on Google Business profiles, and in social media bios. A floating WhatsApp button on websites has been shown to increase customer enquiries significantly.

WhatsApp Link vs WhatsApp Group Link

A WhatsApp link (wa.me) opens a direct chat with a specific number. A WhatsApp group invite link, in contrast, adds the user to a group. Group links are generated inside WhatsApp itself (Group Settings → Invite via Link) and cannot be created by third-party tools.

For business customer service, direct wa.me links are preferred over group links. They allow private, one-on-one conversations that feel personal and professional, rather than adding customers to shared group chats.
